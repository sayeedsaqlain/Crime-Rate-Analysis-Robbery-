# Robbery Rate in Canada-PowerBI

## Intro
In this project I am using the robbery rate data from [StatsCan](https://www.statcan.gc.ca/en/start) website.
I have analyzed the crime rate in different canadian provinces and created dashboard visualization using PowerBI tool.
